# 🇺🇳 Guess The Flag

A simple and colorful flag guessing game built with SwiftUI.  
Test your knowledge of world flags and see how many you can get right out of 8 questions!

## 🧠 Game Mechanics

- You are shown the name of a country.
- You must tap the correct flag out of 3 randomly selected ones.
- If your answer is correct, your score increases.
- If you answer wrong, the correct country's flag is shown.
- The game ends after 8 questions and shows your final score.

## 🔁 Features

- Smooth animations and modern SwiftUI UI
- Gradient background for visual appeal
- Alerts for right/wrong answers
- Final game over alert with reset functionality
- State management using `@State` variables
